# weapon-on-back
Egyszerű fegyver a hátadon script, ox_inventory-hoz.

# Installálás
1. Letöltöd a ZIP fálj-t, kicsomagolod.
2. Berakod a resources mappába a fivem szervereden. Ha bent van editeld a script nevét, töröld ki a végéről ezt: -main
3. Megnyitod a server.cfg fáljt és beleírod valahova az aljára ezt: ensure balint_weapon-on-back
